Input — это элемент для ввода данных пользователем.
Это фактически поле ввода, которое может быть разных типов, например, текстовое поле, пароль, кнопка.

Input Значение
"button" Кликабельная кнопка
"checkbox" Чекбокс (переключатель)
"color" Выбрать цвет
"date" Выбрать дату
"datetime-local" Дату и время выбрать
"email" email-адрес
"file" Выбрать файл
"hidden" Скрытое поле-для хранения данных
"image" Изображение в качестве кнопки отправки формы
"month" Выбрать месяц и год из встроенного календаря
"number" Ограничивает ввод только числами
"password" Для ввода паролей
"radio" Только один вариант из нескольких предложенных.
"range" Для выбора числа в пределах указанного диапазона, используя ползунок.
"reset" Сбрасывает все введенные данные
"search" Для ввода текстовых запросов
"submit" Элемент, который создаёт кнопку отправки формы
"tel" Для ввода телефонного номера
"text" Однострочное текстовое поле для ввода текста
"time" Вводить время в формате часы:минуты
"url" Для ввода адреса веб-страницы
"week" Выбрать год и номер недели
